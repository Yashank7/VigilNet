## 📌 About
VigilNet is a Machine Learning-based Network Intrusion Detection
System that classifies network traffic as Normal or one of
4 attack types — DoS, Probe, R2L, U2R — with 97%+ accuracy.

## 🚀 How to Run
```bash
pip install -r requirements.txt
python nids_model.py
python app.py
```
Then open: http://localhost:5000

## 🤖 Models Used
| Model | Accuracy |
|-------|----------|
| Random Forest | 97.4% |
| SVM | 94.2% |
| KNN | 91.7% |
